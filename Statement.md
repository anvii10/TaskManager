# Personal Task Manager - Statement

## Problem Statement
Managing personal tasks efficiently is difficult without a structured system. Manual tracking leads to missed deadlines and confusion.

## Scope
The project provides a console-based application for users to manage personal tasks with clear workflows and reporting.

## Target Users
- Students
- Professionals
- Anyone who needs to track daily tasks

## High-level Features
- Task creation, editing, and deletion
- Marking tasks as completed
- Viewing tasks by priority, today's tasks, overdue tasks
- Summary of tasks
