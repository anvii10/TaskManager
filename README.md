# Personal Task Manager

## Overview
A console-based task management system in Java allowing users to add, view, edit, delete, and track tasks.

## Features
- Add tasks with title, description, priority, and deadline
- View all tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- View tasks by priority, todayâ€™s tasks, and overdue tasks
- Summary of task counts

## Technologies / Tools
- Java (OOP concepts, Enums, File I/O)
- VS Code / IDE of choice
- Git / GitHub

## Steps to Run
1. Clone the repository.
2. Open terminal inside `src/` folder.  
3. Compile all files. Make sure all .java files are in the same folder.

```javac *.java```  
4. Run the application

```java Main```
